Diccionario git

Comandos más empleados

Git commit -am “..”

cargar y commitear directamente. Con la a no tendríamos que ir fichero por fichero añadiéndolos al área de preparación, solo carga archivos visibles.No sirve en el primer commit

Git log

ver los commit.

Git log - p

ver los commits con las modificaciones que tuvo cada uno.

Git checkout -b nombreRama

crear y cambiarnos a esa rama

Git branch -d nombreRama

eliminar la rama.

Git branch -v

ver todas las ramas y commits a los que apunta cada una.

git remote -v

ver los repositorios en la nube asociados.

git pull dedondelobajamos aqueramalobajamos

bajar los archivos del remoto y mergea.

git fetch dedondelobajamos

bajar la rama que queramos del github o el repositorio entero si lo nombramos. En local creará una rama nueva origin/main que tendremos que mergear manualmente con la master de local.

git clone sshRepositorioremoto

bajar repositorio remoto. Crea una carpeta propia del repositorio la cual ya está asociada al ssh del remoto.
